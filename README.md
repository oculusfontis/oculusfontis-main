# Oculus Fontis Main Website

Official bilingual website for Oculus Fontis and the O.F. Studio Spatial Field series.

## Structure

- `index.html` — standalone main website
- `assets/` — production image assets

## Deployment

This repository is ready for static hosting on Cloudflare Pages.

- Framework preset: None
- Build command: leave blank
- Build output directory: `/`
- Production branch: `main`

The property business remains on the separate subsite: <https://land.oculusfontis.com>.
